# Cyber Risk Quantification Dashboard

This project presents a Cyber Risk Quantification Dashboard built using Power BI to analyze vulnerability data and estimate potential financial exposure from cybersecurity threats.

## Dashboard Preview

![Cyber Risk Dashboard](cyberpulse-dashboard-overview.png)

## Project Overview

This project analyzes cybersecurity vulnerability data and transforms it into a risk intelligence dashboard. By combining CVE identifiers, CVSS scores, attack vectors, and severity levels, the dashboard helps visualize organizational cyber risk exposure and prioritize vulnerability remediation.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI
- GitHub

## Key Insights

- Total vulnerabilities identified: **1,314**
- Average risk score: **5.77**
- Estimated financial exposure: **$185M**
- Network attack vectors represent the largest exploitation pathway
- Critical and high severity vulnerabilities contribute the highest financial risk
